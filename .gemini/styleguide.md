# Gemini Code Assist Style Guide

## 코드 리뷰 언어 규칙

- 모든 코드 리뷰 결과(리뷰 요약, 코멘트, 개선 제안, 승인/반려 사유)는 반드시 한국어(한글)로 작성한다.
- 사용자 요청이나 문맥과 무관하게 코드 리뷰 언어는 항상 한국어로 유지한다.
- 영어 또는 다른 언어로 코드 리뷰 문장을 작성하지 않는다.
- 코드 블록, 파일 경로, 명령어, API/라이브러리 이름 등 기술 고유 표기는 원문 그대로 유지한다.

## 리뷰 작성 원칙

- 핵심 이슈를 먼저 제시하고, 근거와 영향 범위를 한국어로 명확히 설명한다.
- 필요한 테스트/검증 포인트를 한국어로 구체적으로 제시한다.
